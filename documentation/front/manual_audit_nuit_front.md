# MANUEL OPÉRATIONNEL – AUDITEUR DE NUIT (FRONT DESK)

Version complète, chronologique, niveau “Tutoriel pour débutant”.

---

## 0. Logistique dès l’arrivée
- Poste : îlot central (n’importe quel PC).
- Panneau incendie : à gauche de la réception.
- Réserve papier/fournitures : bureau Administration (en bas).
- Code porte admin : `3514#`.

### Wake-up calls (ajout)
1) Vérifier le cahier réception ou demander aux collègues.  
2) Ajout manuel : téléphone le plus à droite → touche “Réveil” → numéro de chambre + `#` → heure (ex. `630` pour 6h30) → choisir AM/PM.

## 1. Préparation des documents (station gauche)
Connexion Windows : User `Auditeur`, MDP `Green176`. Dossier “Audit Pack” sur le bureau (ou clic droit Word/Excel).

### A. Kit comptable & entretien
1) Séparateur daté (Word)  
   - Ouvrir, changer date + jour, imprimer, plier en 2.  
   - Poser à gauche de l’écran (réceptionne les documents comptables).
2) Feuille d’entretien & météo (Word)  
   - Dates J+1 à J+4.  
   - MétéoMédia Laval 7j → Snipping Tool → captures 2 jours par 2 jours.  
   - Word : clic droit image → Change Picture → From Clipboard (garde le format).  
   - Impression obligatoire en recto-verso.
3) Feuille de tournée (Excel)  
   - Changer la date, imprimer, signer.
4) Dépôt : déposer Feuille Entretien + Tournée à l’extrême droite du comptoir (côté tabagie SPESA) pour l’équipe du matin.

### B. Pilotes UPS / FedEx / Cargojet
1) UPS (Chrome, ordi gauche)  
   - Favori “UPS Plugin” → Go to Home → Login → Sign in sheet → View and Print.  
   - UPS → date du lendemain → View → imprimer → plier en 3.
2) FedEx (Excel “FEDEX room records”)  
   - Cahier réception pour les chambres.  
   - Mettre date + chambres (pas de noms). Save + print → plier en 3.
3) Cargojet (Excel “CARGO JET”)  
   - Vérifier dans Lightspeed (par chambre) pour le nom complet du pilote.  
   - Remplir noms + chambres. Save + print.
4) Dépôt final : mettre les 3 feuilles pliées dans le bac réceptionnistes (entre poste droite et centre).

### C. Clés banquets (lendemain)
1) Gros cartable sous l’ordi de droite : relever les événements du lendemain.  
2) Ordi gauche : Word “Copie clef banquet”.  
3) Remplir : affichage, salle, horaire (départ/fin), contact.  
4) Imprimer et placer avec les feuilles pilotes.

## 2. Lancement & rapports (pré-audit)
### A. Lancer Pre-Audit
1) Lightspeed : vérifier qu’il ne reste plus d’Arrivals.  
2) Night Audit (icône lune) → Run Pre-Audit.  
3) Aller chercher les impressions au bureau Admin en bas (code 3514#).

### B. Tri initial & petits soldes
1) Rapports à prendre : Actual Departure Report + Complimentary Rooms Report (poser à droite de l’écran).  
2) Nettoyage balances ±1$ (Actual Departure, colonne Gst Balance) :  
   - Chambre > Modify > Cashiering > Post  
   - Positif (noir) : Dept 40 / Sub 60  
   - Négatif (rouge) : Dept 40 / Sub 10  
   - Ref `CLOSE`, Note `1`, Add to List > Post (solde = 0)  
   - Agrafer le rapport, mettre dans le séparateur daté.

### C. Groupes / Wholesalers
1) Lightspeed > Reports > In-House List.  
2) Group Delegate Guests : page break ON, Room rates YES, Sort Group Code/Room → Submit/Print → agrafer.  
3) Wholesaler Delegates : Tour Code ALL, Sort Room Number → Submit/Print → agrafer.  
4) Classement : chemise bleue datée (tiroir arrière).

### D. Room Status & logbook
1) Lightspeed > Reports > Room Status Details (all rooms, all status, sort room) → Print.  
2) Logbook : photocopier les 3 pages, agrafer sur Room Status.  
3) Déposer en arrière (étagère, sur les caisses) pour Gouvernante.

## 3. Finances & ajustements
### A. Folio Amerispa
1) Cashiering > Amerispa (bleu foncé).  
2) Si solde ≠ 0 : Settle > Apply (0), imprimer folio.  
3) Récupérer les reçus Back, photocopier, agrafer (folio + reçus) → séparateur daté.

### B. Factures Groupes / RL
1) Prendre factures POSitouch du Back → photocopier.  
2) Cashiering > Groupe/RL > imprimer folio.  
3) Agrafer folio devant + reçus derrière → séparateur daté.

### C. Ajustement Internet (Dept 36)
1) Cashier Detail Dept 36, Sub 36.1 (à l’écran).  
2) Par chambre :  
   - Standard : corriger à 4.95 / 0.52 / 0.25.  
   - Elite/Pilote : gratuit 0$ → Transfer vers “Marriott Internet” (sélectionner 3 lignes, glisser).  
   - Non-membre : ne rien changer.  
3) Finalisation : imprimer 36.1 final; folio Marriott Internet Settle 0$ + imprimer; imprimer 36.5; agrafer les 3 → donner au Back.

## 4. Audit (Part 2 & 3)
### A. Avant lancement
1) Vérifier Arrivals.  
2) No-show avec dépôt : Modify > Paiement > Add Deposit (montant HT × 1.19). Si refus : Cancel.

### B. Lancer
1) Night Audit > Post Room and Tax (déconnecte tout, attendre).  
2) Reconnexion → Night Audit > Run Audit.

### C. Pourboires (pendant l’audit)
1) Prendre feuilles individuelles au Back.  
2) Excel POD : saisir ventes et tips par serveur.  
3) Vérif : Total ventes Back – (réception + Spesa) = total ventes POD.  
4) Agrafer POD derrière la feuille du Back → séparateur daté.

---

## 5. Post-audit : impressions & classement
### A. Rapports à imprimer
1) No Posting Allowed : All guests / In-house / By room → pile Restaurant.  
2) Guest List – Charge All : Registered / No comments / Special Service CA / By room → pile Restaurant.  
3) Allotment Overview : Today → Today+49 → pile DBRS.  
4) In-House List G4 : 3 copies (DBRS, Auditeur, Restaurant).  
5) In-House List G4+CL : 2 copies (Superviseur, Restaurant).

Organisation piles :  
- Piles Restaurant à l’arrière (près pile Gouvernante).  
- 2 copies supplémentaires de la pile DBRS complète : Auditeur Back, Réception, Pigeonnier Nathalia (arrière).

### B. High Balance
1) Pour chaque chambre : Cashiering > Settle > Apply.  
2) Si accepté : X rouge (annuler) + “OK”. Si refus : noter. Mettre de côté pour distribution.

### C. No-Shows (Did Not Arrive Open)
1) Modify > Cashiering > Post Dept 1 / Sub 28, montant HT, Ref NO SHOW, Note = nom.  
2) Imprimer folio + feuille d’ajustement (Excel), agrafer (ajustement devant). Bureau réception arrière.  
3) Groupe/RL : poster tout le séjour, transférer au compte maître, dupliquer résa pour nuits restantes (rate 0$).

### D. Check-outs & plug-ins
1) Liste “Départ après Part 2” : check-out (balance 0).  
2) Plug-ins (pilotes) : Quick Modify > Registered > Wholesaler Code (CA*/FE*/UP*) → courriel Outlook (# chambres, heures départ).

### E. Machine à vin (Lounge)
1) Enoclient : Statistics > Reports > Yesterday & Today > Load → imprimer (carré rouge).  
2) POSitouch (Piazza) : Table 800 > Guest 1 > Divers > Vins Ouverts → entrer montant/nom → Pay → Room Charge.  
3) Agrafer les 3 tickets caisse sur la feuille Enoclient → panier factures Back Office.

---

## 6. Fin de quart & nettoyage
1) Courriel Expected Arrivals : Start Today / End Today (template).  
2) Courriel Incidents : envoyer (template).  
3) RESOB : Today → Today+365, imprimer 9 copies (pigeonniers).  
4) Journaux :  
   - Remplacer au Lounge et Spesa.  
   - Gazette de Montréal : garder complet.  
   - Journal de Montréal : garder seulement la couverture.  
   - Le Devoir/autres : jeter.  
   - Stockage : sous le bureau superviseur arrière.
5) Séparer les rapports Part 2 selon la distribution prévue.

Feuilles principales (0-12)

#0 — EJ (Écriture Journal / Grand Livre)

233 lignes × 8 colonnes · Support

Grand livre exporté du système comptable. Chaque ligne = une écriture avec code GL, centre de coûts, description et montant.

ColonneContenu
A (0)Code GL (ex: 075001, 051000, 201802)
B (1)Centre de coûts (CC1)
C (2)Centre de coûts (CC2)
D-E (3-4)Descriptions (libellé comptable)
F (5)Source du document
G (6)Montant ($)

Exemples GL: 075001-075047 (hospitalité), 201802 (ristournes), 202020 (TPS), 202040 (TVH)
